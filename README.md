# Chatbot-AI

Considerando a abrangência dos processos nas áreas de EHA (Exception Handling Area) e Returns (Reversa), foi criada a ferramenta Chatbot, com o objetivo de auxiliar os colaboradores na resolução de dúvidas relacionadas aos processos internos.

O chatbot é integrado a uma IA, que interage com os usuários, fornecendo explicações sobre os processos internos e compartilhando conhecimento operacional.
